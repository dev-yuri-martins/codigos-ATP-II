## Exercicios Lista I

Segue a seguir os exercicio a respeito de funções, procedimentos, visão geral sobre parâmetros de entrada de dados e retorno de dados.

* [Exercicio-01](./exercice01.c) - Crie um procedimento que exiba a mensagem: “Iniciando o sistema”.
* [Exercicio-02](./exercice02.c) - Desenvolva uma função que receba um número inteiro e retorne seu dobro.
* [Exercicio-03](./exercice03.c) - Crie uma função que receba dois números e retorne a soma entre eles.
* [Exercicio-04](./exercice04.c) - Desenvolva um procedimento que receba um nome e exiba: “Olá, [nome]”
* [Exercicio-05](./exercice05.c) - Escreva uma função que receba três notas e retorne a média.
* [Exercicio-06](./exercice06.c) - Crie um procedimento que receba a média de um aluno e informe se ele foi aprovado (média ≥ 6).
* [Exercicio-07](./exercice07.c) - Desenvolva uma função que receba um número e retorne verdadeiro se for par e falso caso contrário.
* [Exercicio-08](./exercice08.c) - Crie um algoritmo que utilize uma função para calcular o quadrado de um número e exiba o resultado.
* [Exercicio-09](./exercice09.c) - Desenvolva um programa modular que: leia dois números, use uma função para calcular a soma e use um procedimento para exibir o resultado
* [Exercicio-10](./exercice10.c) - Crie um conjunto de funções para: calcular a soma, calcular a média e calcular o maior valor. Utilize todas elas em um programa principal.
