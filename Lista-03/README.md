# Exercicios lista 03

Segue a seguir os exercicios a respeito de: recursividade, ponteiros, arrays e strings.

* [Exercicio-01](./l3ex01.c) - Implemente uma função recursiva que calcule o fatorial de um número.
* [Exercicio-02](./l3ex02.c) - Crie uma função recursiva que calcule a soma de 1 até n.
* [Exercicio-03](./l3ex03.c) - Desenvolva uma função recursiva para calcular a potência (base^expoente).
* [Exercicio-04](./l3ex04.c) - Escreva uma função recursiva que conte quantos números existem de n até 0.
* [Exercicio-05](./l3ex05.c) - Crie um programa que utilize ponteiros para alterar o valor de uma variável.
* [Exercicio-06](./l3ex06.c) - Desenvolva um procedimento que receba um ponteiro e dobre o valor apontado.
* [Exercicio-07](./l3ex07.c) - Crie um algoritmo que declare um vetor e altere seus valores usando ponteiros.
* [Exercicio-08](./l3ex08.c) - Desenvolva um programa que receba uma string e altere seu primeiro caractere.
* [Exercicio-09](./l3ex09.c) - Compare (em código) uma função iterativa e uma recursiva para o mesmo problema (ex: soma ou fatorial).
* [Exercicio-10](./l3ex10.c) - Crie um algoritmo que receba um vetor, utilize ponteiros para modificar seus valores e exiba o vetor antes e depois
